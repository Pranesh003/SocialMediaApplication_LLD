# Social Media Console Application (LLD Project)

## Overview
This project is a **console-based Social Media Application** built using **Object-Oriented Programming (OOP)** and **Low-Level Design (LLD)** principles.

The system simulates the core backend functionality of a social media platform where users can:

- Create accounts
- Connect with friends
- Create posts
- View news feeds
- Receive notifications

This project focuses on **backend system design and modular architecture** rather than graphical UI.

---

# Project Objective

The objective of this project is to design and implement a **console-based social networking platform** that allows users to:

- Register profiles
- Connect with friends
- Create and interact with posts
- View a personalized news feed
- Receive system notifications

The project demonstrates **Low Level Design concepts used in real backend systems**.
